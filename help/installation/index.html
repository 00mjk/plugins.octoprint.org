--- layout: page title: Installing a plugin ---
Attention! Plugins are currently only supported in OctoPrint devel version, not the stable master version that is included in OctoPi. In order to be able to use any of the plugins here you'll first need to upgrade to the devel branch if you are not running it already. You can find instructions on how to do so in OctoPrint's FAQ.
All plugins listed in this repository can be installed using OctoPrint's Plugin Manager. or via command line. You have these three options:
  • Open the plugin repository in the Plugin Manager's settings dialog, find the plugin you are interesting in there and install it with the "Install" button provided next to it.
    • {% capture screenie %}{% include canonic_url url='/assets/img/help/install_plugin_from_repo.png' %}{% endcapture %}
  • Copy the part after pip install in the little "Installation" box that's part of every plugin page and paste that into the "Install from URL" text box in the Plugin Manager's settings dialog and hit "Install".
    • {% capture screenie %}{% include canonic_url url='/assets/img/help/install_plugin_from_url_1.png' %}{% endcapture %}
    • {% capture screenie %}{% include canonic_url url='/assets/img/help/install_plugin_from_url_2.png' %}{% endcapture %}
  • Copy the full "Installation" command listed on each plugin page and paste it into the command line of the host you installed OctoPrint on. Note: Make sure you are using the correct version of pip -- on OctoPi you'll need to use ~/oprint/bin/pip instead of just pip.
